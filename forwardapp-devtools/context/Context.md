## 1. Поточне завдання (CURRENT TASK)
- Винести дані для підпису з build.gradle.kts у local.properties (який має бути у .gitignore) 
- Встановити isMinifyEnabled = true для release збірки. Це вимагатиме налаштування proguard-rules.pro, щоб уникнути збоїв через надмірне видалення коду.
- Очистити залежності: Видалити всі дублікати та вирішити конфлікти версій у build.gradle.kts.
- Оновити версію: Змінити versionName на релізну (напр., 1.20.0) та збільшити versionCode.
- Вимкнути бекап через ADB: Встановити android:allowBackup="false", щоб захистити дані користувачів
